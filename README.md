# Authorship_Attribution_R

The aim of this study is to accurately classify the author of a given text. This corpus contains 10 bloggers with 500 data points each, were one data point contains 400-1500 words. The bloggers are well-known in the start-up and venture capital industry and cover topics related to start-ups, innovation, technologies and lifestyle. Data was collected through web-scraping 10 individual blogs. In this corpus, a number of standard stylometric variables were calculated, such as word length, vocabulary diversity, and word frequency, and n-grams including bigrams and trigrams.
